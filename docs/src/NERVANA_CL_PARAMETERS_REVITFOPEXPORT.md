# Экспорт Revit ФОП

*Доступно с версии 1.0.1*

Позволяет сохранить все параметры БСК в текстовый файл общих параметров Autodesk Revit аналогично базовой команде "Файл ➝ Экспорт ➝ Экспортировать параметры в XML"

## Принцип работы

В диалоге сохранения файлов Пользователь указывает место и имя сохраняемого текстового файла. Запускается процедура преобразования определений параметров CADLib в представления параметров по логике Revit ФОП:

- категория (группа) параметра CADLib преобразуется в отдельную Группу (GROUP) Revit ФОП;

- Name параметра Revit ФОП -- это внутреннее имя определения параметра CADLib;

- Description параметра Revit ФОП -- это отображаемое имя определения параметра CADLib;

- Visible параметра Revit ФОП -- это обратное свойство видимости категории параметров CADLib;

- UserModifiable параметра Revit ФОП -- это обратное свойство Readonly определения параметра CADLib;

Функция стабильно отрабатывает на "3.0.572.2976"
