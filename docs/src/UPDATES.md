# История обновлений плагина Nervana

# Версия 1.0.1

13.01.2026

**Добавлено**:

- Поддержка версий nanoCAD 22 и старше для плагинов к платформе nanoCAD и ModelStudio COM;

- Новый плагин с функциями для CADLib: Модель и Архив;

- [Nervana NBIMS] Новые функции в плагин для nanoCAD BIM Строительство - утилиты: "Расставить отверстия", "Концептуальный рельеф", "Обозреватель конструктивных сборок"; работа с помещениями: "Авто-контур", "Расставить в помещениях", "Перекрытия по помещениям", "Помещения по перекрытиям", "Помещения по контурам", "Связать стены с помещением", "Связать объекты с помещением"; аналитические команды: "Построить тени от Солнца";

- [Nervana COM] команда "Обозреватель параметров";

**Изменено**:

* Механика установки плагина (для каждого плагина своя лента команд), отдельные версии для старых версии nanoCAD;
* [Nervana NC] Новые окна обозревателей свойств теперь вызываются как модальные для nanoCAD (не будут перекрываться nanoCAD'ом);

# Версия 1.0.0

23.12.2025
