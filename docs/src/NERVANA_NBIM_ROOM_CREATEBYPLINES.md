# Помещения по контурам

*Доступно с версии 1.0.1*

Создает помещения на основе контуров замкнутых полилиний.

Ожидает выбора полилиний. Можно выбирать не только полилинии (будут использованы только объекты полилиний, если полилиния не замкнутая, то она не будет использована в качестве контура для создания помещения).
