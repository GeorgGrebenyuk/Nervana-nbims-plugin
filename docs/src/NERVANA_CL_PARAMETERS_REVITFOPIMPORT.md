# Импорт Revit ФОП

*Доступно с версии 1.0.1*

Позволяет импортировать определения параметров из текстового файла общих параметров (ФОП) Autodesk Revit.

Есть 2 реализации команды -- настоящая и с суффиксом (2). Первый вариант пытается создавать определения параметров `CLibParamDefInfo` для каждого параметра Revit, а вторая записывает информацию по параметрам в XML-файл, представляющий собой форму записи параметров БСК в XML, а затем пытается импортировать его в CADLib командой `CADLibrary.ImportParameters()`. 

❗На сборке "3.0.572.2976" оба способа сваливают CADLib в фатал из-за внутренних исключений в реализации API.

## Принцип работы

Пользователь выбирает текстовый (txt) файл, представляющий перечень общих параметров по форме, принятой в Autodesk Revit. Если файл не получилось считать, то выбросится исключение, а выполнение функции прервется

> Файл Revit ФОП не был успешно обработан!. Функция будет прервана

Новые определения параметров создаются без привязки к конкретным категориям объектов CADLib (TODO: сделать мэппирование типов).
